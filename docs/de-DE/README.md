Apache Doris ist eine Open-Source-Datenbank für Echtzeitanalysen und Suche, die auf einer MPP-Architektur basiert. Sie bietet schnelle SQL-Analysen, beschleunigte Lakehouse-Abfragen und hybride Suche über strukturierte Daten, Textdaten und Vektordaten hinweg.

Auf der [offiziellen Website](https://doris.apache.org/) finden Sie die aktuelle Produktübersicht, Anwendungsfälle, Updates zum Ökosystem, Blogs und Anwenderberichte. Versionsupdates finden Sie in [allen Versionshinweisen](https://doris.apache.org/releases/all-release).

## 📈 Anwendungsfälle

| Anwendungsfall | Was er bietet |
| -------- | ---------------- |
| [Analysen für externe Kunden](https://doris.apache.org/use-cases/customer-facing-analytics) | Stellen Sie externen Nutzern interaktive Analysen mit Antwortzeiten unter einer Sekunde bereit. |
| [Data Warehousing](https://doris.apache.org/use-cases/data-warehousing) | Bauen Sie ein einziges Echtzeit-Warehouse über verschiedene Geschäftsbereiche hinweg auf. |
| [Observability](https://doris.apache.org/use-cases/observability) | Analysieren Sie Protokolle, Ereignisse und Metriken mit hohem Durchsatz per SQL. |
| [Doris für AI](https://doris.apache.org/use-cases/ai) | Nutzen Sie Vektor-, Text-, JSON- und strukturierte Suche in einer SQL-Engine. |

## 🚀 Kernfunktionen

Apache Doris ist auf drei Kernfähigkeiten ausgerichtet. Die Website ist die maßgebliche Quelle für detaillierte Produktbeschreibungen und Beispiele.

| Fähigkeit | Was sie bietet |
| ---------- | ---------------- |
| [Echtzeitanalysen](https://doris.apache.org/#real-time-analytics) | Streaming-Ingestion, inkrementelle Transformationen und Abfragen mit Antwortzeiten unter einer Sekunde bei hoher Nebenläufigkeit. |
| [Lakehouse-Analysen](https://doris.apache.org/#lakehouse-analytics) | Schnelle SQL-Analysen über offene Tabellenformate wie Iceberg, Delta Lake und Hudi. |
| [Hybride Suche](https://doris.apache.org/#hybrid-search) | SQL-native Analysen über JSON-, Volltext- und Vektordaten für AI- und Such-Workloads. |

## 🔌 Ökosystem

Doris steht im Zentrum des modernen Daten-Stacks. Es verbindet vorgelagerte Datenbanken, Streaming-Systeme und Lakehouse-Speicher mit nachgelagerten BI-, AI-, Analyse- und Observability-Tools.

## 🧱 Architektur

Apache Doris unterstützt Deployments mit gekoppeltem Compute und Storage ebenso wie Deployments mit entkoppeltem Compute und Storage. Im entkoppelten Modus laufen zustandslose Compute-Gruppen auf gemeinsam genutztem Objektspeicher. So können Sie Compute nach Bedarf skalieren und Workloads isolieren.
