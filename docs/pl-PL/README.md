Apache Doris to otwartoźródłowa baza danych do analityki i wyszukiwania w czasie rzeczywistym, zbudowana na architekturze MPP. Zapewnia szybką analitykę SQL, przyspieszanie zapytań lakehouse oraz wyszukiwanie hybrydowe w danych strukturalnych, tekstowych i wektorowych.

Odwiedź [oficjalną stronę internetową](https://doris.apache.org/), aby poznać najnowszy opis produktu, przypadki użycia, aktualizacje ekosystemu, blogi i historie użytkowników. Informacje o wersjach znajdziesz w [pełnych informacjach o wydaniach](https://doris.apache.org/releases/all-release).

## 📈 Przypadki użycia

| Przypadek użycia | Co oferuje |
| -------- | ---------------- |
| [Analityka dla użytkowników końcowych](https://doris.apache.org/use-cases/customer-facing-analytics) | Udostępniaj użytkownikom zewnętrznym interaktywną analitykę z czasem odpowiedzi poniżej sekundy. |
| [Hurtownia danych](https://doris.apache.org/use-cases/data-warehousing) | Buduj jedną hurtownię czasu rzeczywistego obejmującą różne obszary biznesowe. |
| [Obserwowalność](https://doris.apache.org/use-cases/observability) | Analizuj za pomocą SQL wysokoprzepustowe logi, zdarzenia i metryki. |
| [Doris dla AI](https://doris.apache.org/use-cases/ai) | Korzystaj z wyszukiwania wektorowego, tekstowego, JSON i strukturalnego w jednym silniku SQL. |

## 🚀 Główne możliwości

Apache Doris opiera się na trzech kluczowych możliwościach. Strona internetowa jest źródłem prawdy dla szczegółowych opisów produktu i przykładów.

| Możliwość | Co oferuje |
| ---------- | ---------------- |
| [Analityka w czasie rzeczywistym](https://doris.apache.org/#real-time-analytics) | Strumieniowe wczytywanie danych, transformacje przyrostowe i zapytania poniżej sekundy przy wysokiej współbieżności. |
| [Analityka Lakehouse](https://doris.apache.org/#lakehouse-analytics) | Szybka analityka SQL na otwartych formatach tabel, takich jak Iceberg, Delta Lake i Hudi. |
| [Wyszukiwanie hybrydowe](https://doris.apache.org/#hybrid-search) | Natywna dla SQL analityka danych JSON, pełnotekstowych i wektorowych dla obciążeń AI oraz wyszukiwania. |

## 🔌 Ekosystem

Doris znajduje się w centrum nowoczesnego stosu danych. Łączy źródłowe bazy danych, systemy strumieniowe i magazyny lakehouse z narzędziami BI, AI, analityki i obserwowalności.

## 🧱 Architektura

Apache Doris obsługuje wdrożenia ze sprzężonymi oraz rozdzielonymi warstwami obliczeń i pamięci masowej. W trybie rozdzielonym bezstanowe grupy obliczeniowe działają na współdzielonym magazynie obiektowym, dzięki czemu można skalować obliczenia na żądanie i izolować obciążenia.

<p align="center">
  <img src="https://doris.apache.org/images/next/home-page/cs-decoupled.jpg" alt="Apache Doris compute-storage decoupled architecture" width="850" />
</p>

Więcej informacji znajdziesz w [przewodniku wdrażania](https://doris.apache.org/docs/dev/install/intro) oraz [przewodniku po trybach wdrożenia](https://doris.apache.org/docs/dev/install/choosing-deployment-mode).
