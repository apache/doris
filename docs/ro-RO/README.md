Apache Doris este o bază de date open-source pentru analiză și căutare în timp real, construită pe arhitectura MPP. Oferă analiză SQL rapidă, accelerarea interogărilor lakehouse și căutare hibridă în date structurate, text și date vectoriale.

Explorează [site-ul oficial](https://doris.apache.org/) pentru cea mai recentă prezentare a produsului, cazuri de utilizare, noutăți din ecosistem, bloguri și povești ale utilizatorilor. Pentru actualizări de versiune, consultă [toate notele de lansare](https://doris.apache.org/releases/all-release).

## 📈 Cazuri de utilizare

| Caz de utilizare | Ce oferă |
| -------- | ---------------- |
| [Analiză pentru clienți](https://doris.apache.org/use-cases/customer-facing-analytics) | Livrează analize interactive sub o secundă către utilizatori externi. |
| [Data Warehousing](https://doris.apache.org/use-cases/data-warehousing) | Construiește un singur depozit de date în timp real pentru toate domeniile de business. |
| [Observabilitate](https://doris.apache.org/use-cases/observability) | Analizează cu SQL volume mari de loguri, evenimente și metrici. |
| [Doris pentru AI](https://doris.apache.org/use-cases/ai) | Folosește căutare vectorială, text, JSON și căutare structurată într-un singur motor SQL. |

## 🚀 Capabilități principale

Apache Doris este construit în jurul a trei capabilități principale. Site-ul web este sursa de adevăr pentru descrieri detaliate ale produsului și exemple.

| Capabilitate | Ce oferă |
| ---------- | ---------------- |
| [Analiză în timp real](https://doris.apache.org/#real-time-analytics) | Ingestie streaming, transformare incrementală și interogări sub o secundă la concurență ridicată. |
| [Analiză Lakehouse](https://doris.apache.org/#lakehouse-analytics) | Analiză SQL rapidă peste formate de tabele deschise, precum Iceberg, Delta Lake și Hudi. |
| [Căutare hibridă](https://doris.apache.org/#hybrid-search) | Analiză nativă SQL peste date JSON, full-text și vectoriale pentru sarcini AI și de căutare. |

## 🔌 Ecosistem

Doris se află în centrul stivei moderne de date. Conectează baze de date upstream, sisteme de streaming și stocare lakehouse cu instrumente downstream de BI, AI, analiză și observabilitate.

## 🧱 Arhitectură

Apache Doris acceptă atât implementări cu calcul și stocare cuplate, cât și implementări cu calcul și stocare decuplate. În modul decuplat, grupurile de calcul fără stare rulează peste stocare de obiecte partajată, astfel încât poți scala calculul la cerere și poți izola sarcinile de lucru.

<p align="center">
  <img src="https://doris.apache.org/images/next/home-page/cs-decoupled.jpg" alt="Apache Doris compute-storage decoupled architecture" width="850" />
</p>

Află mai multe în [ghidul de implementare](https://doris.apache.org/docs/dev/install/intro) și în [ghidul pentru modul de implementare](https://doris.apache.org/docs/dev/install/choosing-deployment-mode).
