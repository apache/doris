Apache Doris adalah database analitik dan pencarian real-time open-source yang dibangun di atas arsitektur MPP. Apache Doris menyediakan analitik SQL yang cepat, akselerasi kueri lakehouse, serta pencarian hibrida di data terstruktur, teks, dan vektor.

Jelajahi [situs web resmi](https://doris.apache.org/) untuk melihat ringkasan produk terbaru, kasus penggunaan, pembaruan ekosistem, blog, dan cerita pengguna. Untuk pembaruan versi, lihat [semua catatan rilis](https://doris.apache.org/releases/all-release).

## 📈 Kasus Penggunaan

| Kasus Penggunaan | Yang disediakan |
| -------- | ---------------- |
| [Analitik untuk Pelanggan](https://doris.apache.org/use-cases/customer-facing-analytics) | Hadirkan analitik interaktif subdetik kepada pengguna eksternal. |
| [Data Warehousing](https://doris.apache.org/use-cases/data-warehousing) | Bangun satu warehouse real-time untuk berbagai domain bisnis. |
| [Observabilitas](https://doris.apache.org/use-cases/observability) | Analisis log, event, dan metrik throughput tinggi dengan SQL. |
| [Doris untuk AI](https://doris.apache.org/use-cases/ai) | Gunakan pencarian vektor, teks, JSON, dan data terstruktur dalam satu mesin SQL. |

## 🚀 Kemampuan Inti

Apache Doris dibangun di sekitar tiga kemampuan inti. Situs web resmi menjadi sumber utama untuk deskripsi produk dan contoh yang terperinci.

| Kemampuan | Yang disediakan |
| ---------- | ---------------- |
| [Analitik Real-Time](https://doris.apache.org/#real-time-analytics) | Ingestion streaming, transformasi inkremental, dan kueri subdetik pada konkurensi tinggi. |
| [Analitik Lakehouse](https://doris.apache.org/#lakehouse-analytics) | Analitik SQL cepat di atas format tabel terbuka seperti Iceberg, Delta Lake, dan Hudi. |
| [Pencarian Hibrida](https://doris.apache.org/#hybrid-search) | Analitik native SQL untuk data JSON, full-text, dan vektor bagi beban kerja AI dan pencarian. |

## 🔌 Ekosistem

Doris berada di pusat stack data modern. Doris menghubungkan database upstream, sistem streaming, dan penyimpanan lakehouse dengan alat downstream untuk BI, AI, analitik, dan observabilitas.

## 🧱 Arsitektur

Apache Doris mendukung deployment compute-storage coupled dan compute-storage decoupled. Dalam mode decoupled, grup komputasi stateless berjalan di atas object storage bersama, sehingga Anda dapat menskalakan komputasi sesuai kebutuhan dan mengisolasi beban kerja.

<p align="center">
  <img src="https://doris.apache.org/images/next/home-page/cs-decoupled.jpg" alt="Apache Doris compute-storage decoupled architecture" width="850" />
</p>

Pelajari lebih lanjut di [panduan deployment](https://doris.apache.org/docs/dev/install/intro) dan [panduan mode deployment](https://doris.apache.org/docs/dev/install/choosing-deployment-mode).
