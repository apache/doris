Apache Doris je odprtokodna podatkovna zbirka za analitiko in iskanje v realnem času, zgrajena na arhitekturi MPP. Omogoča hitro SQL analitiko, pospeševanje poizvedb v lakehouse okoljih in hibridno iskanje po strukturiranih, besedilnih in vektorskih podatkih.

Na [uradnem spletnem mestu](https://doris.apache.org/) najdete najnovejši pregled izdelka, primere uporabe, novosti v ekosistemu, bloge in uporabniške zgodbe. Za posodobitve različic glejte [vse opombe ob izdajah](https://doris.apache.org/releases/all-release).

## 📈 Primeri uporabe

| Primer uporabe | Kaj omogoča |
| -------- | ---------------- |
| [Analitika za uporabnike](https://doris.apache.org/use-cases/customer-facing-analytics) | Zunanjim uporabnikom ponudite interaktivno analitiko z odzivnim časom pod sekundo. |
| [Podatkovno skladišče](https://doris.apache.org/use-cases/data-warehousing) | Zgradite eno podatkovno skladišče v realnem času za različna poslovna področja. |
| [Opazljivost](https://doris.apache.org/use-cases/observability) | S SQL analizirajte velike količine dnevnikov, dogodkov in metrik. |
| [Doris za AI](https://doris.apache.org/use-cases/ai) | Uporabite vektorsko, besedilno, JSON in strukturirano iskanje v enem SQL pogonu. |

## 🚀 Ključne zmožnosti

Apache Doris temelji na treh ključnih zmožnostih. Uradno spletno mesto je izvor resnice za podrobne opise izdelka in primere.

| Zmožnost | Kaj omogoča |
| ---------- | ---------------- |
| [Analitika v realnem času](https://doris.apache.org/#real-time-analytics) | Pretočni vnos, postopno preoblikovanje podatkov in poizvedbe z odzivnim časom pod sekundo pri visoki sočasnosti. |
| [Lakehouse analitika](https://doris.apache.org/#lakehouse-analytics) | Hitra SQL analitika nad odprtimi formati tabel, kot so Iceberg, Delta Lake in Hudi. |
| [Hibridno iskanje](https://doris.apache.org/#hybrid-search) | SQL-native analitika nad JSON, polnim besedilom in vektorskimi podatki za delovne obremenitve AI in iskanja. |

## 🔌 Ekosistem

Doris je v središču sodobnega podatkovnega sklada. Povezuje izvorne podatkovne zbirke, pretočne sisteme in lakehouse shrambo z odjemnimi orodji za BI, AI, analitiko in opazljivost.

## 🧱 Arhitektura

Apache Doris podpira uvedbe, pri katerih sta računanje in shramba povezana, ter uvedbe, pri katerih sta ločena. V ločenem načinu se brezstanjne računske skupine izvajajo nad skupno objektno shrambo, zato lahko računanje prilagajate na zahtevo in izolirate delovne obremenitve.

<p align="center">
  <img src="https://doris.apache.org/images/next/home-page/cs-decoupled.jpg" alt="Apache Doris compute-storage decoupled architecture" width="850" />
</p>

Več preberite v [vodniku za uvedbo](https://doris.apache.org/docs/dev/install/intro) in [vodniku za izbiro načina uvedbe](https://doris.apache.org/docs/dev/install/choosing-deployment-mode).
