# SHOW TABLET
## description
    该语句用于显示指定tablet id 信息（仅管理员使用）
    语法：
        SHOW TABLET tablet_id


## example
        // 显示指定 tablet id 为 10000 的 tablet 的父层级 id 信息
        SHOW TABLET 10000;

## keyword
    SHOW,TABLET,LIMIT
