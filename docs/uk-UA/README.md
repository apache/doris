Apache Doris є аналітичною та пошуковою базою даних з відкритим кодом для роботи в реальному часі, побудованою на архітектурі MPP. Вона забезпечує швидку SQL-аналітику, прискорення запитів до lakehouse та гібридний пошук у структурованих, текстових і векторних даних.

Перегляньте [офіційний вебсайт](https://doris.apache.org/), щоб знайти актуальний огляд продукту, сценарії використання, оновлення екосистеми, блоги та історії користувачів. Інформацію про оновлення версій дивіться в [усіх примітках до випусків](https://doris.apache.org/releases/all-release).

## 📈 Сценарії використання

| Сценарій використання | Що він надає |
| -------- | ---------------- |
| [Аналітика для клієнтських продуктів](https://doris.apache.org/use-cases/customer-facing-analytics) | Доставляйте зовнішнім користувачам інтерактивну аналітику з відгуком менш ніж за секунду. |
| [Сховище даних](https://doris.apache.org/use-cases/data-warehousing) | Створюйте єдине сховище даних у реальному часі для різних бізнес-доменів. |
| [Спостережуваність](https://doris.apache.org/use-cases/observability) | Аналізуйте високонавантажені журнали, події та метрики за допомогою SQL. |
| [Doris для AI](https://doris.apache.org/use-cases/ai) | Використовуйте векторний, текстовий, JSON і структурований пошук в одному SQL-рушії. |

## 🚀 Основні можливості

Apache Doris побудовано навколо трьох основних можливостей. Вебсайт є джерелом правди для докладних описів продукту та прикладів.

| Можливість | Що вона надає |
| ---------- | ---------------- |
| [Аналітика в реальному часі](https://doris.apache.org/#real-time-analytics) | Потокове завантаження даних, інкрементальні перетворення та запити з відгуком менш ніж за секунду за високої паралельності. |
| [Lakehouse-аналітика](https://doris.apache.org/#lakehouse-analytics) | Швидка SQL-аналітика відкритих табличних форматів, зокрема Iceberg, Delta Lake і Hudi. |
| [Гібридний пошук](https://doris.apache.org/#hybrid-search) | Нативна для SQL аналітика JSON, повнотекстових і векторних даних для AI та пошукових навантажень. |

## 🔌 Екосистема

Doris розташована в центрі сучасного стеку даних. Вона поєднує висхідні бази даних, потокові системи та lakehouse-сховища з низхідними інструментами BI, AI, аналітики й спостережуваності.

## 🧱 Архітектура

Apache Doris підтримує розгортання як із поєднаними обчисленнями та сховищем, так і з відокремленими обчисленнями та сховищем. У відокремленому режимі безстанові обчислювальні групи працюють поверх спільного об'єктного сховища, тому ви можете масштабувати обчислення на вимогу та ізолювати робочі навантаження.

<p align="center">
  <img src="https://doris.apache.org/images/next/home-page/cs-decoupled.jpg" alt="Apache Doris compute-storage decoupled architecture" width="850" />
</p>

Докладніше читайте в [посібнику з розгортання](https://doris.apache.org/docs/dev/install/intro) і [посібнику з вибору режиму розгортання](https://doris.apache.org/docs/dev/install/choosing-deployment-mode).
