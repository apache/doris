# CANCEL ALTER
Description
This statement is used to undo an ALTER operation.
1. 撤销 ALTER TABLE COLUMN 操作
Grammar:
CANCEL ALTER TABLE COLUMN
FROM db_name.table_name

2. 撤销 ALTER TABLE ROLLUP 操作
Grammar:
CANCEL ALTER TABLE ROLLUP
FROM db_name.table_name
