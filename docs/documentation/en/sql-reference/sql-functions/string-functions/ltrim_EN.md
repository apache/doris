'35; ltrim
Description
'35;'35;' 35; Syntax

'VARCHAR ltrim (VARCHAR str)'


Remove the space that appears continuously from the beginning of the parameter str

'35;'35; example

```
mysql > SELECT ltrim (ab d');
+------------------+
1244; ltrim (ab d') 1244;
+------------------+
1.2.4.;b d
+------------------+
```
