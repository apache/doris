# strleft
Description
'35;'35;' 35; Syntax

'WARCHAR STRAIGHT (WARCHAR STR)'


It returns the left part of a string of specified length

'35;'35; example

```
mysql> select strleft("Hello doris",5);
+------------------------+
(Hello doris', 5)'1244;
+------------------------+
| Hello                  |
+------------------------+
```
##keyword
STRLEFT
