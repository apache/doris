It35; ASCII
Description
'35;'35;' 35; Syntax

'INT AXES (WARCHAR STR)'


Returns the ASCII code corresponding to the first character of the string

'35;'35; example

```
mysql > select ascii ('1');
+------------+
124; ASCII ('1') 124;
+------------+
|         49 |
+------------+

mysql > select axes ('234');
+--------------+
124; ASCII ('234') 124;
+--------------+
|           50 |
+--------------+
```
##keyword
ASCII
