# from_days
## Description
### Syntax

`DATE FROM_DAYS(INT N)`


Calculate which day by the number of days from 0000-01-01

## example

```
mysql > select from u days (730669);
+-------------------+
| from_days(730669) |
+-------------------+
| 2000-07-03        |
+-------------------+
