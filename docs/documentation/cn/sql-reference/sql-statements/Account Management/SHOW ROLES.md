# SHOW ROLES
## description
    该语句用于展示所有已创建的角色信息，包括角色名称，包含的用户以及权限。

    语法：
        SHOW ROLES;
    
## example

    1. 查看已创建的角色：

        SHOW ROLES;

## keyword
    SHOW,ROLES
