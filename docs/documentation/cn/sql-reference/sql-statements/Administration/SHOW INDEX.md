# SHOW INDEX

## description

    该语句用于展示一个表中索引的相关信息，目前只支持bitmap 索引
    语法：
        SHOW INDEX[ES] FROM [db_name.]table_name;

## example

    1. 展示指定 table_name 的下索引
        SHOW INDEX FROM example_db.table_name;

## keyword

    SHOW,INDEX
