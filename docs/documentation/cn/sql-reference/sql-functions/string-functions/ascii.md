# ascii

## Description

返回字符串第一个字符串对应的ascii 码

## Syntax

`INT ascii(VARCHAR)`

## Examples

```
mysql> select ascii('1');
+------------+
| ascii('1') |
+------------+
|         49 |
+------------+

mysql> select ascii('234');
+--------------+
| ascii('234') |
+--------------+
|           50 |
+--------------+
```
