Apache Doris é um banco de dados de código aberto para análise e busca em tempo real, criado sobre uma arquitetura MPP. Ele oferece análises SQL rápidas, aceleração de consultas em lakehouse e busca híbrida em dados estruturados, textuais e vetoriais.

Explore o [site oficial](https://doris.apache.org/) para ver a visão geral mais recente do produto, casos de uso, atualizações do ecossistema, blogs e histórias de usuários. Para atualizações de versão, consulte [todas as notas de lançamento](https://doris.apache.org/releases/all-release).

## 📈 Casos de uso

| Caso de uso | O que oferece |
| -------- | ---------------- |
| [Análises para usuários finais](https://doris.apache.org/use-cases/customer-facing-analytics) | Entregue análises interativas em menos de um segundo para usuários externos. |
| [Armazenamento de dados](https://doris.apache.org/use-cases/data-warehousing) | Crie um data warehouse em tempo real que atenda a vários domínios de negócio. |
| [Observabilidade](https://doris.apache.org/use-cases/observability) | Analise logs, eventos e métricas de alto volume com SQL. |
| [Doris para AI](https://doris.apache.org/use-cases/ai) | Use buscas vetoriais, textuais, JSON e estruturadas em um único mecanismo SQL. |

## 🚀 Principais capacidades

Apache Doris é desenvolvido em torno de três capacidades principais. O site é a fonte de referência para descrições detalhadas do produto e exemplos.

| Capacidade | O que oferece |
| ---------- | ---------------- |
| [Análises em tempo real](https://doris.apache.org/#real-time-analytics) | Ingestão por streaming, transformação incremental e consultas em menos de um segundo sob alta concorrência. |
| [Análises em lakehouse](https://doris.apache.org/#lakehouse-analytics) | Análises SQL rápidas em formatos de tabela abertos, como Iceberg, Delta Lake e Hudi. |
| [Busca híbrida](https://doris.apache.org/#hybrid-search) | Análises nativas em SQL sobre JSON, texto completo e dados vetoriais para cargas de trabalho de AI e busca. |

## 🔌 Ecossistema

Doris fica no centro da pilha de dados moderna. Ele conecta bancos de dados de origem, sistemas de streaming e armazenamento lakehouse a ferramentas de destino para BI, AI, análises e observabilidade.

## 🧱 Arquitetura

Apache Doris oferece suporte a implantações com computação e armazenamento acoplados ou desacoplados. No modo desacoplado, grupos de computação sem estado são executados sobre armazenamento de objetos compartilhado, permitindo escalar a computação sob demanda e isolar cargas de trabalho.
