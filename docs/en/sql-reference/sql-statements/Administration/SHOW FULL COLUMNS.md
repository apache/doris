# SHOW FULL COLUMNS
## description
    This statement is used to view some information about columns of a table.

    Syntax:
        SHOW FULL COLUMNS FROM tbl;
        
## example

    1. View the column information of specified table

        SHOW FULL COLUMNS FROM tbl;
        
## keyword
    
    SHOW,FULL,COLUMNS
