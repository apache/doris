# SHOW TABLET
## Description
    This statement is used to display information of the specified tablet (for administrators only)
    Grammar:
        SHOW TABLET tablet_id

## example

        // Display parent level ID information of tablet with specified tablet id of 10000
        SHOW TABLET 10000;

## keyword
SHOW,TABLET
