# SHOW TABLE ID
## Description
This statement is used to display database name, table name according to table id (for administrators only)
Grammar:
SHOW TABLE [table_id]

## example
1. Display database name, table name according to table id
SHOW TABLE 10001;

## keyword
SHOW,TABLE
