# DROP ENCRYPTKEY

## Description

### Syntax

```
DROP ENCRYPTKEY key_name
```

### Parameters

> `key_name`: 要删除密钥的名字, 可以包含数据库的名字。比如：`db1.my_key`。

删除一个自定义密钥。密钥的名字完全一致才能够被删除

## example

1. 删除掉一个密钥

```
DROP ENCRYPTKEY my_key;
```

## keyword

    DROP,ENCRYPTKEY
