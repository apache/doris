# SHOW CREATE ROUTINE LOAD
## description
    该语句用于展示例行导入作业的创建语句
    语法：
        SHOW [ALL] CREATE ROUTINE LOAD for load_name;
        
    说明：
       `ALL`: 可选参数，代表获取所有作业，包括历史作业
       `load_name`: 例行导入作业名称

## example
    1. 展示默认db下指定例行导入作业的创建语句
        SHOW CREATE ROUTINE LOAD for test_load

## keyword
    SHOW,CREATE,ROUTINE,LOAD
