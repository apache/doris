# PAUSE ROUTINE LOAD
## example

1. 暂停名称为 test1 的例行导入作业。

    PAUSE ROUTINE LOAD FOR test1;

2. 暂停所有正在运行的例行导入作业

    PAUSE ALL ROUTINE LOAD;

## keyword
    PAUSE,ALL,ROUTINE,LOAD
