# SHOW REMOTE STORAGES

## Description

    该语句用于查看当前存在的远端存储
    语法：
        SHOW REMOTE STORAGES;

    说明：
        1. Name：远端存储的名字
        2. Type：远端存储的类型
        3. Properties：远端存储的参数
        
## keyword

    SHOW, REMOTE STORAGES
