## SHOW-CATALOGS

### Name

SHOW CATALOGS

### Description

该语句用于显示已存在是数据目录（catalog）

语法：

```sql
SHOW CATALOGS
```

返回结果说明：

* CatalogId：数据目录唯一ID
* CatalogName：数据目录名称。其中 internal 是默认内置的 catalog，不可修改。
* Type：数据目录类型。

### Example

1. 查看当前已创建的数据目录

   ```sql
   SHOW CATALOGS;
	+-----------+-------------+----------+
	| CatalogId | CatalogName | Type     |
	+-----------+-------------+----------+
	|     10024 | hive        | hms      |
	|         0 | internal    | internal |
	+-----------+-------------+----------+
   	```
