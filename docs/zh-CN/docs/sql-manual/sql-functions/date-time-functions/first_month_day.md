## first_month_day
### Description
#### Syntax

`DATE first_month_day(DATETIME date)`

返回输入日期中月份的第一天；所以返回的日期中，年和月不变，日置为01

### example

```
mysql > select first_month_day('2021-12-22');
+-------------------+
| first_month_day('2021-12-22 00:00:00') |
+-------------------+
| 2021-12-01        |
+-------------------+
```
