## last_day
### Description
#### Syntax

`DATE last_day(DATETIME date)`

返回输入日期中月份的最后一天；所以返回的日期中，年和月不变，日可能是如下情况：
'28'(非闰年的二月份), 
'29'(闰年的二月份),
'30'(四月，六月，九月，十一月),
'31'(一月，三月，五月，七月，八月，十月，十二月)

### example

```
mysql > select last_day('2000-02-03');
+-------------------+
| last_day('2000-02-03 00:00:00') |
+-------------------+
| 2000-02-29        |
+-------------------+
```
