Apache Doris, MPP mimarisi üzerine kurulu açık kaynaklı, gerçek zamanlı bir analiz ve arama veritabanıdır. Hızlı SQL analitiği, lakehouse sorgu hızlandırması ve yapılandırılmış, metin ve vektör verileri üzerinde hibrit arama sağlar.

En güncel ürün özeti, kullanım senaryoları, ekosistem güncellemeleri, bloglar ve kullanıcı hikayeleri için [resmi web sitesini](https://doris.apache.org/) inceleyin. Sürüm güncellemeleri için [tüm sürüm notlarına](https://doris.apache.org/releases/all-release) bakın.

## 📈 Kullanım Senaryoları

| Kullanım Senaryosu | Ne sağlar |
| -------- | ---------------- |
| [Müşteriye Yönelik Analitik](https://doris.apache.org/use-cases/customer-facing-analytics) | Dış kullanıcılara saniyenin altında yanıt veren etkileşimli analitik sunun. |
| [Veri Ambarı](https://doris.apache.org/use-cases/data-warehousing) | İş alanları genelinde tek bir gerçek zamanlı ambar oluşturun. |
| [Gözlemlenebilirlik](https://doris.apache.org/use-cases/observability) | Yüksek hacimli günlükleri, olayları ve metrikleri SQL ile analiz edin. |
| [AI için Doris](https://doris.apache.org/use-cases/ai) | Tek bir SQL motorunda vektör, metin, JSON ve yapılandırılmış aramayı kullanın. |

## 🚀 Temel Yetenekler

Apache Doris üç temel yetenek etrafında tasarlanmıştır. Ayrıntılı ürün açıklamaları ve örnekler için başvuru kaynağı web sitesidir.

| Yetenek | Ne sağlar |
| ---------- | ---------------- |
| [Gerçek Zamanlı Analitik](https://doris.apache.org/#real-time-analytics) | Yüksek eşzamanlılık altında akış verisi alımı, artımlı dönüşüm ve saniyenin altında sorgular. |
| [Lakehouse Analitiği](https://doris.apache.org/#lakehouse-analytics) | Iceberg, Delta Lake ve Hudi gibi açık tablo formatları üzerinde hızlı SQL analitiği. |
| [Hibrit Arama](https://doris.apache.org/#hybrid-search) | AI ve arama iş yükleri için JSON, tam metin ve vektör verileri üzerinde SQL'e özgü analitik. |

## 🔌 Ekosistem

Doris modern veri yığınının merkezinde yer alır. Yukarı akış veritabanlarını, akış sistemlerini ve lakehouse depolamasını aşağı akış BI, AI, analitik ve gözlemlenebilirlik araçlarına bağlar.

## 🧱 Mimari

Apache Doris hem hesaplama ve depolamanın birlikte çalıştığı hem de birbirinden ayrıldığı dağıtımları destekler. Ayrıştırılmış modda, durum bilgisi tutmayan hesaplama grupları paylaşılan nesne depolama üzerinde çalışır. Böylece hesaplamayı isteğe göre ölçeklendirebilir ve iş yüklerini yalıtabilirsiniz.
