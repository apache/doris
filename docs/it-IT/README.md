Apache Doris è un database open source per analisi e ricerca in tempo reale, costruito su architettura MPP. Offre analisi SQL rapide, accelerazione delle query lakehouse e ricerca ibrida su dati strutturati, testuali e vettoriali.

Visita il [sito web ufficiale](https://doris.apache.org/) per la panoramica più recente del prodotto, i casi d'uso, gli aggiornamenti dell'ecosistema, i blog e le storie degli utenti. Per gli aggiornamenti di versione, consulta [tutte le note di rilascio](https://doris.apache.org/releases/all-release).

## 📈 Casi d'uso

| Caso d'uso | Cosa offre |
| -------- | ---------------- |
| [Analisi per utenti finali](https://doris.apache.org/use-cases/customer-facing-analytics) | Distribuisce analisi interattive con latenza inferiore al secondo agli utenti esterni. |
| [Data warehousing](https://doris.apache.org/use-cases/data-warehousing) | Crea un unico warehouse in tempo reale per più domini aziendali. |
| [Osservabilità](https://doris.apache.org/use-cases/observability) | Analizza log, eventi e metriche ad alto throughput con SQL. |
| [Doris per AI](https://doris.apache.org/use-cases/ai) | Usa ricerca vettoriale, testuale, JSON e strutturata in un unico motore SQL. |

## 🚀 Funzionalità principali

Apache Doris si basa su tre funzionalità principali. Il sito web è la fonte di riferimento per descrizioni dettagliate del prodotto ed esempi.

| Funzionalità | Cosa offre |
| ---------- | ---------------- |
| [Analisi in tempo reale](https://doris.apache.org/#real-time-analytics) | Ingestione streaming, trasformazione incrementale e query con latenza inferiore al secondo ad alta concorrenza. |
| [Analisi lakehouse](https://doris.apache.org/#lakehouse-analytics) | Analisi SQL rapide su formati di tabelle aperti come Iceberg, Delta Lake e Hudi. |
| [Ricerca ibrida](https://doris.apache.org/#hybrid-search) | Analisi native SQL su dati JSON, full-text e vettoriali per carichi di lavoro AI e di ricerca. |

Doris si colloca al centro del moderno data stack. Collega database upstream, sistemi di streaming e storage lakehouse con strumenti downstream di BI, AI, analisi e osservabilità.

## 🧱 Architettura

Apache Doris supporta distribuzioni con compute e storage accoppiati e distribuzioni con compute e storage disaccoppiati. In modalità disaccoppiata, gruppi di compute stateless operano su object storage condiviso, così puoi scalare il compute on demand e isolare i carichi di lavoro.

<p align="center">
  <img src="https://doris.apache.org/images/next/home-page/cs-decoupled.jpg" alt="Apache Doris compute-storage decoupled architecture" width="850" />
