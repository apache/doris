Apache Doris は、MPP アーキテクチャ上に構築された、オープンソースのリアルタイム分析および検索データベースです。高速な SQL 分析、レイクハウスクエリの高速化、構造化データ、テキスト、ベクトルデータを横断するハイブリッド検索を提供します。

最新の製品概要、ユースケース、エコシステムの更新情報、ブログ、ユーザーストーリーについては、[公式ウェブサイト](https://doris.apache.org/)をご覧ください。バージョン更新については、[すべてのリリースノート](https://doris.apache.org/releases/all-release)を参照してください。

## 📈 ユースケース

| ユースケース | 提供するもの |
| -------- | ---------------- |
| [顧客向け分析](https://doris.apache.org/use-cases/customer-facing-analytics) | 外部ユーザーにサブ秒単位のインタラクティブ分析を提供できます。 |
| [データウェアハウス](https://doris.apache.org/use-cases/data-warehousing) | ビジネス領域を横断する単一のリアルタイムウェアハウスを構築できます。 |
| [オブザーバビリティ](https://doris.apache.org/use-cases/observability) | 高スループットのログ、イベント、メトリクスを SQL で分析できます。 |
| [AI 向け Doris](https://doris.apache.org/use-cases/ai) | 1 つの SQL エンジンで、ベクトル、テキスト、JSON、構造化検索を利用できます。 |

## 🚀 コア機能

Apache Doris は 3 つのコア機能を中心に構築されています。詳細な製品説明と例については、公式ウェブサイトを正としてください。

| 機能 | 提供するもの |
| ---------- | ---------------- |
| [リアルタイム分析](https://doris.apache.org/#real-time-analytics) | ストリーミング取り込み、増分変換、高い同時実行性のもとでのサブ秒クエリを提供します。 |
| [レイクハウス分析](https://doris.apache.org/#lakehouse-analytics) | Iceberg、Delta Lake、Hudi などのオープンテーブル形式に対して高速な SQL 分析を実行できます。 |
| [ハイブリッド検索](https://doris.apache.org/#hybrid-search) | AI と検索ワークロード向けに、JSON、全文、ベクトルデータを横断する SQL ネイティブの分析を提供します。 |

## 🔌 エコシステム

Doris はモダンデータスタックの中心に位置します。上流のデータベース、ストリーミングシステム、レイクハウスストレージを、下流の BI、AI、分析、オブザーバビリティツールと接続します。

## 🧱 アーキテクチャ

Apache Doris は、コンピュートとストレージを結合したデプロイと、コンピュートとストレージを分離したデプロイの両方をサポートします。分離モードでは、ステートレスなコンピュートグループが共有オブジェクトストレージ上で動作するため、必要に応じてコンピュートをスケールし、ワークロードを分離できます。

<p align="center">
  <img src="https://doris.apache.org/images/next/home-page/cs-decoupled.jpg" alt="Apache Doris compute-storage decoupled architecture" width="850" />
</p>

詳しくは、[デプロイガイド](https://doris.apache.org/docs/dev/install/intro)と[デプロイモードガイド](https://doris.apache.org/docs/dev/install/choosing-deployment-mode)をご覧ください。
