## 2. T07a — DELETE/MERGE sink 方言（连接器本地、dormant、UT 可测、0 fe-core 改）

## 3. T07b — O5-2 生产半（fe-core 通用 target-only 抽取 → `ConnectorPredicate` → `applyWriteConstraint`）

> 连接器消费半 T05 已就位（`IcebergConnectorTransaction.applyWriteConstraint` 暂存中立谓词 + commit 时 `buildWriteConstraintExpression` 经 `IcebergPredicateConverter` 惰性转）。T07b 建 fe-core 生产半。legacy 3-hop 侧通道（command→executor.setConflictDetectionFilter→txn）**保活**（`IcebergDeleteExecutor.java:92`/`IcebergMergeExecutor.java:80` live 直到 P6.6）。

### 3.1 通用 target-only 抽取（拆 `IcebergConflictDetectionFilterUtils`）

新 fe-core 通用工具（如 `datasource.WriteConstraintExtractor`）= 拆 legacy `IcebergConflictDetectionFilterUtils.buildConflictDetectionFilter`（`:60-81`）的**通用收集半**：
- 递归 `LogicalFilter` 合取遍历（移植 `collectTargetConjuncts:83-96`），`IcebergExternalTable` 参数仅用于 `getId()` → 改 `long targetTableId`（连接器无关）。
- slot-origin 检查（移植 `isTargetOnlyPredicate:98-123`）：每个 slot 须 `SlotReference` 且 `slot.getOriginalTable().get().getId() == targetTableId`。

**🔴 合成列排除（critic finding 1 = BLOCKER，证伪 synthesis）**：`$row_id` slot 由 `SlotReference.fromColumn(...,ICEBERG_ROWID_COL,...)`（`IcebergNereidsUtils.java:171`）建，`fromColumn` 置 `originalColumn = column`（`SlotReference.java:158-162`）⟹ `getOriginalColumn().isPresent()` 返回 true、且 `originalTable == 目标表` ⟹ **`isPresent()` 捷径排不掉 `$row_id`、会让其通过整个 target-only 判定 → 冲突 filter 漂移**。
**修**：合成列排除用 **iceberg 变换侧提供的 `Predicate<SlotReference>`**（复刻 legacy 按名排除 `Column.ICEBERG_ROWID_COL` + `IcebergMetadataColumn.isMetadataColumn`，`IcebergConflictDetectionFilterUtils.java:111,114`），由 `RowLevelDmlTransform` 注入通用抽取器。`IcebergMetadataColumn`（pkg `datasource.iceberg`）引用留在 iceberg 变换内、不进通用 util（import-gate 意图 + 命名洁净）。

### 3.2 nereids `Expression` → `ConnectorExpression` 转换（真正的 fork）

实证：fe-core **无** nereids→ConnectorExpression 转换器（唯一 `ExprToConnectorExpressionConverter.convert` 吃 legacy `analysis.Expr`，line 28/91）。两路（T07b recon 时定，记 [OQ-T07b]）：
- **Option A**：新写 `NereidsToConnectorExpressionConverter`（nereids `Expression`→`ConnectorExpression`）。干净、无 `PlanTranslatorContext`；须**精确**匹配 `ExprToConnectorExpressionConverter` 的值/类型编码契约（`IcebergPredicateConverter` 依赖之，`IcebergPredicateConverterTest:72`），否则下推漂移。
- **Option B**：nereids→`Expr`（`ExpressionTranslator.translate(Expression, PlanTranslatorContext)`，`:146`）→ `ExprToConnectorExpressionConverter`。无重复，但须命令期（`planner.plan()` 后）拿到 slot-bound `PlanTranslatorContext`，把 target-only SlotReference 绑进新 context 非平凡。
- **倾向 A**（避免 context 复活的脆弱性）；T07b recon 实测 B 的 context 可达性后定。

### 3.3 字节 parity 转换对齐（用户裁定 = 完整 parity；critic findings 2/3）

连接器 `IcebergPredicateConverter`（T02 为 scan 造）与 legacy `convertPredicateToIcebergExpression`（`IcebergConflictDetectionFilterUtils.java`）的冲突检测路径**对齐**：
- **同列 OR-guard（critic finding 2 = HIGH 正确性）**：legacy `buildOr` 经 `isSameColumnPredicate`（`:147-149`）丢「两臂引不同列」的 OR；连接器 `buildOr`（`IcebergPredicateConverter.java:162-172`）**无**此 guard → 保留跨列 OR **收窄**冲突 filter → **可能漏真实并发冲突 = 丢更新（不安全）**。修=连接器 `buildOr` 加同列 guard（或抽取器不向 `applyWriteConstraint` 发跨列 OR）。
- **IS NULL / BETWEEN（critic finding 3）**：legacy 处理 `IsNull`(:159)、`Between`(:165)；连接器 `build`（`:120-138`）落空 → `return null` 丢。补连接器 `buildIsNull`/`buildBetween`。
- **`Not` 限制**：legacy 仅 `Not(IsNull)`（`:152-157`）；连接器 `buildNot`（`:174-177`）处理一般 `Not(child)` → 比 legacy 宽。补限制为 `Not(IsNull)`。
- ⚠️ 风险：`IcebergPredicateConverter` 同时服务 scan 下推（T02），改之须**不破坏 scan parity**。**实现策略**：若 scan/conflict 的 legacy 目标分叉（scan=`convertToIcebergExpr`、conflict=`convertPredicateToIcebergExpression`），用**独立冲突检测路径/模式 flag** 而非盲扩共享 build()（T07b recon 时核两 legacy 矩阵差异）。

### 3.4 接线 = 3-hop 换 1-hop（dormant）

壳的 planner-drive loop 调 `transform.extractWriteConstraint(analyzedPlan, targetTableId)` → 中立 `ConnectorPredicate` → `connectorTransaction.applyWriteConstraint(pred)`（`ConnectorTransaction.java:116-118` default no-op；`IcebergConnectorTransaction:239-242` 暂存、commit 时惰性转）。**plan 时调、begin/commit 前**（匹配暂存-惰性转）。legacy 3-hop 保活直到 P6.6。

`WriteConstraintExtractor` round-trip：合成 analyzed plan → `ConnectorPredicate` → `IcebergConnectorTransaction.buildWriteConstraintExpression` → iceberg `Expression` **== legacy `buildConflictDetectionFilter` 输出**（含 `$row_id` 排除、跨列 OR-guard、IS NULL/BETWEEN/Not(IsNull) parity 逐 case）。nereids→ConnectorExpression（Option A）oracle 网格镜像 `IcebergPredicateConverterTest`，断值/类型编码 == `ExprToConnectorExpressionConverter` 契约。
